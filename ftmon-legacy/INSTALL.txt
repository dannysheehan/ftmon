To Install
----------
For all help/suggestions (no matter how small) email dsheehan@ftmon.org


QUICK
-----
cd /opt
zcat /var/tmp/ftmonV2.tar.gz | tar -cvf -
cd /opt/ftmon/base/lib/linux
./install.sh
service ftmon start
Open your web browser to /var/ftmon/html/index.html


FOR OTHER THAN /opt INSTALLATION
--------------------------------

1. cd ftmon/lib/linux

2. edit sysconfig_linux to the paths you want to install the base and cfg
directories under 

Default is
  /opt/ftmon/base
  /opt/ftmon/cfg

3. sh ./install.sh

4. Move the files
mv ftmon /opt/ftmon

5. service ftmon start

6. Open your web browser to /var/ftmon/html/index.html
